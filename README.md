\# 🎤 Voice Gender Classification (Hybrid CNN + LSTM)



\## 📌 Project Overview

This project classifies human voice as Male or Female using a Hybrid CNN + LSTM deep learning model trained on audio features.



\---



\## ⚙️ Features

\- Upload WAV audio file

\- 3-second audio chunking

\- MFCC feature extraction

\- Hybrid CNN + LSTM prediction

\- Confidence score output



\---



\## 🧠 Model Details

\- CNN: Extracts audio features

\- LSTM: Learns temporal patterns

\- Hybrid Model Accuracy: 99.76%



\---



\## 🚀 Deployment

Built using Streamlit and deployed on Streamlit Cloud.



\---



\## 📊 Output

\- Male / Female classification

\- Confidence percentage

\- Chunk-based prediction
